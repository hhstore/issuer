# issuer


- github issues blog with theme.
- 基于 github issue blog, 生成 github pages 的博客页面工具.

## Features:

- [ ] github issue to blog
- [ ] 支持特殊模板指令
- [ ] 支持评论功能
- [ ] 支持 markdown 扩展. 


## Online:

- https://hhstore.github.io/issuer
- 在线访问


## Quickstart


- flutter 2.5.1
- 基于 Taskfile 脚本

- install: 


```bash

# install: 
brew install go-task/tap/go-task


# init:
task install

# gen files:
task init




```




- run web:


```bash

task run:web

```

- after change, generate files:

```bash
# gen files:
task gen

```

## Deploy:


- gen files:


```bash

task deploy


```
